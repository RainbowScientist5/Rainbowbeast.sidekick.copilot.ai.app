# Security Policy

## Supported Versions

We release patches for security vulnerabilities. Which versions are supported and receiving security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us by emailing security@example.com. We will respond as quickly as possible to address the issue.

Please include as much information as possible to help us resolve the issue quickly.
